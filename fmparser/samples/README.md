# Samples

Place controlled `.fmf` experiments here. Each sample pair should differ by exactly one tactical
change so the differential analyser can build reliable byte-to-setting evidence.

Example sequence:

- `sample01-balanced.fmf`
- `sample02-positive.fmf`
- `sample03-higher-tempo.fmf`
- `sample04-lower-tempo.fmf`
- `sample05-4-2-3-1.fmf`
- `sample06-4-3-3.fmf`
