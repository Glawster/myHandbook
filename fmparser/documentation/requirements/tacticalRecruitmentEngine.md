## Acceptance criteria

- [ ] A tactical profile can be loaded from YAML.
- [ ] Attribute weights do not need to total 100.
- [ ] The application normalises valid weights before scoring.
- [ ] Missing attributes are shown clearly rather than treated as zero.
- [ ] Minimum thresholds can be overridden by league.
- [ ] Players are ranked by tactical-fit score.
- [ ] A candidate can be compared with the current first-choice player.
- [ ] The calculation has automated tests.
